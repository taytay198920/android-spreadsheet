# 🚀 一键部署指南

## 方法一：GitHub Actions自动构建（推荐）

这是最简单的方法，无需本地环境：

### 步骤：
1. **创建GitHub仓库**
   - 访问 [github.com](https://github.com)
   - 点击右上角"+" → "New repository"
   - 仓库名：`android-spreadsheet`

2. **上传代码**
   ```bash
   cd /root/.openclaw/workspace/android_spreadsheet
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/android-spreadsheet.git
   git push -u origin main
   ```

3. **触发构建**
   - 进入GitHub仓库页面
   - 点击 "Actions" 标签
   - 等待构建完成（约5-10分钟）

4. **下载APK**
   - 构建完成后，点击 "android-spreadsheet-apk"
   - 下载 `app-release.apk` 文件

## 方法二：使用在线Flutter IDE

### FlutLab.io
1. 访问 [FlutLab.io](https://flutlab.io)
2. 注册/登录账号
3. 点击 "New Project" → "Import from Git"
4. 粘贴GitHub仓库URL
5. 点击 "Build" → "Build Android APK"

### CodeMagic.io
1. 访问 [CodeMagic.io](https://codemagic.io)
2. 连接GitHub账号
3. 选择你的仓库
4. 点击 "Start new build"
5. 配置为Android构建

## 方法三：本地构建（需要安装环境）

如果你决定安装Flutter环境：

### 快速安装脚本：
```bash
# 安装Flutter
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:`pwd`/flutter/bin"

# 安装Android SDK（简化版）
flutter doctor --android-licenses

# 构建APK
cd /root/.openclaw/workspace/android_spreadsheet
flutter pub get
flutter build apk --release

# APK位置
ls build/app/outputs/flutter-apk/
```
